Quran Reference Linker

A single-file JavaScript widget that detects Quran references in Islamic articles and blog posts, then shows a hover/tap tooltip with Arabic text, English translation, and recitation audio.

Where To Use It

  • Islamic blogs and article archives
  • WordPress, Ghost, Hugo, Jekyll, Next.js, static sites
  • Long-form essays with many Quran references
  • HTML article pages generated from editorial CMS content

Integration Notes

  • Add the script once in the site-wide article layout or theme header.
  • The widget auto-scans <p>, <li>, <span>, and <div>.
  • It ignores refs inside links and code blocks.
  • It also watches for late-added content, so SPA or CMS-injected sections still work.

PDF Limitation

This works on HTML pages, not inside raw .pdf files served as browser PDF documents. If a PDF is converted into HTML or rendered through a custom HTML text layer, then the widget can be used there too.

Network / CSP

  • connect-src https://api.quran.com
  • media-src https://verses.quran.com
  • style-src 'unsafe-inline' for the current injected shadow-DOM styles
